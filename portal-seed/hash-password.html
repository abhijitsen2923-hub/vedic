Hash a password for the Students sheet

Local utility — runs entirely in your browser. Nothing is sent over the network. Use this to convert a plaintext password into the PBKDF2 format the password_hash column needs. (Tip: the sheet's IVA → Set / reset password menu does this in-place — this page is the offline fallback.)

— Hash will appear here —

How to use:

  1. Type the password you'd like the student to log in with.
  2. Click Hash it. The hash appears below (starts with pbkdf2$).
  3. Click Copy hash.
  4. Open the Students tab of the IVA Portal DB Google Sheet and paste into the password_hash column of a new row.
  5. Share the plaintext password with the student privately — don't save it anywhere yourself.

Why: The portal compares PBKDF2 hashes server-side, so the Sheet stores only the hash. It is one-way — even with the hash, the plaintext can't be recovered (a forgotten password is reset, not recovered).
